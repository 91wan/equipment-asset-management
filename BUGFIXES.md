## 快速记录

```bash
# 在项目根目录运行
python3 ../../.openclaw/scripts/log-bugfix.py \
  --title "Bug简短描述" \
  --severity 中等 \
  --description "问题描述" \
  --root-cause "根因分析" \
  --fix-process "修复步骤" \
  --changes "- 文件: xxx.py" \
  --commit
```
